# SaaS Template

## Stack

Auth - Supabase
Billing - Stripe
Database - PostgreSQL (Supabase)
ORM - Drizzle

## General setup

Setup pre-commit hooks with husky
In the root of the project run:

```bash
pnpm install

pnpm run prepare
```
